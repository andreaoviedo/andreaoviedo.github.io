Estructura Primaria

Como ya revisamos, las etiquetas de la estructura primaria son:

  • <html> define el documento html completo.
  • <head> contiene elementos especiales que le indican al navegador dónde encontrar hojas de estilo, proporcionar meta data y más.
  • <body> contiene el contenido del documento (lo que se muestra dentro de la ventana del navegador).

Elementos de la etiqueta <head>

Dentro de la etiqueta <head>, se suelen incluir las siguientes etiquetas:

<title> define el título del documento, es lo que aparece en el título de la pestaña o la ventana del navegador. Es solamente texto.

<head>
            <title>A Meaningful Page Title</title> <!--Este título está muy relacionado con el posicionamiento en buscadores-->
        </head>

<link> define la relación entre el documento actual y un recurso externo, por ejemplo un archivo de CSS o favicon (la imagen que aparece junto al título de la página)

<link rel="stylesheet" href="mystyle.css">
        <link rel="icon" type="image/png" href="img/favicon.png">

<style> se utiliza para establecer los estilos CSS de la página actual, esta etiqueta se ve a detalle en 🎨Introducción a CSS

<meta> especifica el conjunto de caracteres, la descripción de la página, las palabras clave, el autor del documento y la configuración de la ventana gráfica. Ejemplos de su uso:

  • Definir el tipo de caracteres que se usa en la página, permitiendo que se desplieguen correctamente acentos y ñ's.
    <meta charset="UTF-8">
  • Describir el contenido de la página también para buscadores
    <meta name="description" content="Free Web tutorials">
  • Definir palabras clave para posicionamiento en buscadores
    <meta name="keywords" content="HTML, CSS, JavaScript">
  • Definir el autor de la página
    <meta name="author" content="John Doe">
  • Establecer las características de la ventana gráfica para que la página se vea correctamente en todos los dispositivos.
    <meta name="viewport" content="width=device-width, initial-scale=1.0">

Referencia w3schools: HTML Head

Tomando en cuenta los elementos anteriores, te recomiendo que todos tus documentos HTML incluyan como mínimo la siguiente estructura:

<!DOCTYPE html>
        <html>
            <head>
                <title>Título de la página</title>
                        <link rel="icon" type="image/png" href="img/favicon.png" />
                <meta charset="UTF-8">
            </head>
            <body>
                <!-- Aquí van todos los elementos HTML que utilices -->
            </body>
        </html>

Para publicar tu página, es muy recomendable que también incluyan los tags de meta mencionados anteriormente.

Elementos de bloque

Un elemento a nivel de bloque siempre comienza en una nueva línea y ocupa todo el ancho disponible (se extiende hacia la izquierda y hacia la derecha tanto como puede).

<p> Párrafos

La etiqueta <p></p> se utiliza para distribuir el texto en párrafos. Un párrafo siempre comienza en una nueva línea y los navegadores agregan automáticamente un espacio en blanco (un margen) antes y después de un párrafo.

Referencia w3schools: Paragraphs

<br> Salto de línea

Inserta un salto de línea sencillo en el texto.

Referencia w3schools: <br> tag

<h1> Encabezados

Existen 6 títulos y subtítulos que se despliegan dentro de la página. Se definen con las etiquetas <h1> a <h6>

Referencia w3schools: Headers

Listas

<ol> Listas ordenadas

Una lista ordenada comienza con la etiqueta <ol>. Cada elemento de la lista comienza con la etiqueta <li>.

Los elementos de la lista se marcan con números de forma predeterminada.

Su atributo más común es type, su valor determinará el tipo de enumeración

1 = números (es el valor por defecto)

A = letras mayúsculas

a = letras minúsculas

I = números romanos en mayúsculas

i = números romanos en minúsculas

Otro atributo común es start, indica en qué posición se inicia la numeración. Por defecto es 1, prueba en los diferentes tipos de lista agregar start="2"

Referencia w3schools: Ordered Lists

<ul> Listas desordenadas

Las listas desordenadas funcionan exactamente igual que las ordenadas, con la diferencia de que los elementos se despliegan con viñetas en lugar de numeraciones.

Referencia w3schools: Unordered Lists

<div> Divisiones

Se utiliza para definir secciones del documento HTML. Funciona como un contenedor de otros elementos.

Permite que se le asignen estilos y comportamientos que revisaremos a detalle en los siguientes capítulos.

Referencia w3schools: <div> tag

Elementos en línea

Un elemento en línea comienza en la misma línea y solo ocupa el ancho necesario.

<a> Enlaces

La etiqueta <a> permite definir un hipervínculo, su atributo más importante es href pues es el que define el destino del enlace.

Otro de sus atributos más comunes es target, este permite definir si la página se abrirá en el mismo lugar o en otra pestaña.

Si aún no conoces el enlace, puedes utilizar la almohadilla (#) para hacer que el texto sea "clickeable", este rederigirá a la misma página.

Referencia w3schools: Links

<img> Imágenes

Se utiliza para incrustar una imagen en una página web. Su atributo más importante es src, en donde se especifica la ruta del servidor donde se encuentra la imagen o una URL externa.

Otro de sus atributos importantes es alt, este especifica un texto que describe la imagen, si el navegador no encuentra la imagen desplegará este texto en su lugar. También es útil con fines de accesibilidad y de posicionamiento (SEO).

width y height permiten establecer el ancho y el alto, sin embargo, veremos más adelante que es una mejor práctica establecerlos con CSS.

Referencia w3schools: Images

<iframe> Marcos incorporados

Se utilizan para incrustar un contenido de una página dentro de otra. Muchos sitios incluyen la opción de Embed (Incustrar contenido) para que puedas hacerlo más fácilmente, por ejemplo los videos de YouTube

Referencia w3schools: iframes

<span> Contenedor en línea

Es un contenedor en línea que se utiliza para marcar una parte de un texto o una parte de un documento, normalmente es útil para dar un estilo diferente o atribuirle un comportamiento con JavaScript, lo cual veremos más a detalle en los siguientes capítulos.

Referencia w3schools: <span> tag

Formato

<em>(emphasis) y <i>(italic) se utilizan para que la letra sea itálica.

<strong>(strong emphasis) y <b>(black) se utilizan para que la letra sea negrita.

Citas de texto

<q> y <blockquote> permiten dar formato a las citas de texto. <q> agrega comillas a la cita y <blockquote> identa el texto. Esto puede verse de forma diferente dependiendo el navegador.

Referencia w3schools: Quotations

Entidades

Algunos caracteres están reservados en HTML. Si usas los signos menor que (<) o mayor que (>) en su texto, el navegador puede interpretarlos como etiqueta y generar errores o no desplegarse.

Las entidades de caracteres se utilizan para mostrar caracteres reservados en HTML.

Referencia: Full Character Entity Reference

Emojis

Puedes incluir emojis en tus páginas web, solo debes conocer su código decimal o hexadecimal. Existen muchas hojas de referencia, yo te recomiendo usar Full List of Emojis 2020. Encuentra el emoji que te guste y agrega antes de su código &#x y después de este punto y coma ; Por ejemplo:

Se escribiría como: &#x1F929;

Referencia w3schools Emoji Unicode

Iconos

También puedes incluir en tus documentos HTML diferentes iconos. Existen varias librerías, una de las más populares es Font Awesome. Para utilizarla incluye el siguiente código dentro de la etiqueta <head>

<link
              rel="stylesheet"
              href="https://pro.fontawesome.com/releases/v5.10.0/css/all.css"
              integrity="sha384-AYmEC3Yw5cVb3ZcuHtOA93w35dYTsvhLPVnYs9eStHfGJvOvKxVfELGroGkvsg+p"
              crossorigin="anonymous"
            />

Deberás buscar el icono que quieres en Icons y copiar el código que se verá algo como:

<i class="fas fa-heart"></i>

Prácticas

  • Nivel Aprendiz

    Realiza tu CV en HTML

    1. Asegúrate de incluir por lo menos:
    • Nombre completo
    • Titular
    • Fotografía
    • Semblanza o perfil profesional
    • Educación (últimos 2 niveles académicos)
    • Experiencia laboral (mínimo 2 trabajos): Si no has trabajado puedes incluir algún proyecto en el que hayas participado, un servicio social, intercambio, etc.
    • Habilidades y conocimiento
    • Datos de contacto: correo, alguna red social, ubicación (solo ciudad)

    2. Emplea todo lo que has aprendido sobre HTML

    • Estructura básica del documento HTML
    • Por lo menos las etiquetas: p, h1, h2, h3, h4, a, img, iframe, br, ul, li, em, strong
    • No olvides incluir 1 icono, 1 emoji, ponerle favicon a la página y que el título de la misma (en la etiqueta meta) sea tu nombre o CV + tu nombre.

    3. Ve un poco más allá

    • Incluye una sublista
    • Haz que al dar clic en tu email se abra la app de correo y tenga un mensaje listo para ser enviado.
    • Investiga cómo hacer una tabla y pon algún contenido ahí, por ejemplo la información de contacto.
